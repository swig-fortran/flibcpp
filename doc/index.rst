Flibcpp
=======

.. MAIN MATTER
.. ***************************************************************************

.. raw:: latex

   \begin{abstract}

Flibcpp uses SWIG-Fortran to generate native Fortran-2003 interfaces to
efficient and robust algorithms and data containers implemented in the C++
standard library.
